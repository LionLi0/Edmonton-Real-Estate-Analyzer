## Overview
This tool analyzes real estate data from various neighborhoods in Edmonton (e.g., Strathcona, Downtown). It processes raw property records to calculate:
- **Average Market Price** per neighborhood.
- **Price-to-Area Ratios** (Price/SqFt).
- **Vacancy Rates** and property availability.

## Key Features
- **Data Parsing:** Efficiently reads and structures raw text data.
- **Statistical Analysis:** Utilizes Python dictionaries (hash maps) for O(1) data aggregation.
- **Formatted Reporting:** Generates clean, user-friendly CLI tables for decision making.

## Tech Stack
- Python 3.x (Standard Library)
- File I/O
- Data Structures (Lists, Dictionaries)
